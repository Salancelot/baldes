Detailed Description

A class representing a Splay Tree, which is a self-adjusting binary search tree.

The Splay Tree supports efficient insertion, deletion, and search operations by performing splay operations that move accessed nodes closer to the root, thereby improving access times for frequently accessed nodes.

The tree nodes store intervals [low, high] and a bucket index associated with each interval.
